# Foundation - Public Distribution

**Foundation** is a local-first AI agent orchestration platform with 125+ specialized skills.

## 🚀 Quick Start

1. **Download** `Foundation-Setup.exe` from [Releases](https://github.com/EmmanuelOrtiz87/foundation-public/releases)
2. **Run** the installer (accepts EULA)
3. **Open** terminal and type: `wf health`
4. **Read** `docs/guides/GETTING-STARTED.md` for usage
