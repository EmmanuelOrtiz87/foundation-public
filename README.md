## ¿Qué es Foundation?

Foundation es un **stack de desarrollo potenciado por IA** que transforma tu IDE en un equipo completo de ingeniería. No es un chat, no es un plugin — es un **orquestador inteligente** que entiende tu código, tu arquitectura y tu flujo de trabajo.

```
Tú dices:  "implementar login con JWT"
Foundation: → Analiza tu stack → Diseña la solución → Escribe el código → 
             → Ejecuta tests → Valida seguridad → Entrega resultado
```

---

## ¿Por qué es diferente?

| En lugar de... | Foundation hace... |
|---------------|-------------------|
| Promptear una y otra vez | Auto-delega a 16 agentes especializados |
| Perder contexto entre chats | Memoria persistente (Engram) entre sesiones |
| Archivos de documentación muertos | Skills vivas que ejecutan tareas reales |
| Dependencias externas y APIs de pago | 100% local, zero vendor lock-in |

## ⚡ Quick Start

```powershell
# Opción 1 — Descargar instalador
#   Descarga Foundation-Setup.exe desde Releases

# Opción 2 — Bootstrap directo
.\scripts\foundation\bootstrap.ps1

# Usar
wf start-session     # Inicia sesión con memoria persistente
wf judgment-day      # Verifica que todo está OK (14 gates)
wf version           # Muestra versión y skills disponibles
```

> 💡 **Requisito único**: PowerShell 7+ y Git 2.50+.

## 🏛️ Cómo funciona

```
Tú (IDE/CLI) → wf.ps1 → Auto-delegation → Agente especializado → Skill MCP → Resultado
                         ↓
                    Engram (memoria persistente entre sesiones)
```

1. **Escribes tu tarea** en lenguaje natural
2. **Foundation analiza** qué se necesita (código, testing, diseño, DevOps...)
3. **Delega al agente** especializado con la skill adecuada
4. **Ejecuta y valida** — tests, security, quality gates
5. **Persiste en memoria** — la próxima sesión recuerda contexto
