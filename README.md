## What is Foundation?

Foundation is an **AI-powered development stack** that turns your IDE into a full engineering team. Not a chat, not a plugin — it's an **intelligent orchestrator** that understands your code, your architecture, and your workflow.

```
You say:    "implement login with JWT"
Foundation: → Analyzes your stack → Designs the solution → Writes code →
             → Runs tests → Validates security → Delivers result
```

---

## Why is it different?

| Instead of... | Foundation does... |
|---------------|-------------------|
| Prompting over and over | Auto-delegates to 16 specialized agents |
| Losing context between chats | Persistent memory (Engram) across sessions |
| Dead documentation files | Living skills that execute real tasks |
| External dependencies and paid APIs | 100% local, zero vendor lock-in |

## ⚡ Quick Start

```powershell
# 1. Download Foundation.exe from the root of this repository
# 2. Right-click → Run as Administrator (NSIS installer wizard)
# 3. Place master.key in C:\Program Files\Foundation\keys\
# 4. Use the Desktop or Start Menu shortcut

# Once installed, from the terminal:
wf start-session     # Start session with persistent memory
wf judgment-day      # Verify everything is OK (14 gates)
wf version           # Show version and available skills
```

> 💡 **Requirements**: PowerShell 7+, Git 2.50+, and a **master.key** (request access to the private repository).
>
> 📥 **Download**: `Foundation.exe` is at the root of this repository — a single installer with a professional wizard.

## 🏛️ How it works

```
You (IDE/CLI) → wf.ps1 → Auto-delegation → Specialized agent → Skill MCP → Result
                         ↓
                    Engram (persistent memory across sessions)
```

1. **Write your task** in natural language
2. **Foundation analyzes** what's needed (code, testing, design, DevOps...)
3. **Delegates to the specialized agent** with the right skill
4. **Executes and validates** — tests, security, quality gates
5. **Persists to memory** — next session remembers context
